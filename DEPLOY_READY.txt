🚀 HOW TO DEPLOY:

Option 1: One-Click Deploy (Recommended)
────────────────────────────────────────
1. python setup.py                 # Install dependencies
2. Edit backend/config/.env        # Add GROQ_API_KEY
3. python deploy.py                # Start everything
4. Open http://localhost:8000

Option 2: Manual Deploy
───────────────────────
Terminal 1:
  cd backend
  python server.py

Terminal 2:
  cd frontend/web/public
  python serve.py

Then open: http://localhost:8000

⚙️ CONFIGURATION NEEDED:

1. Get Groq API Key:
   - Visit: https://console.groq.com
   - Create free account
   - Copy API key
   - Paste into backend/config/.env

2. Add Contacts:
   - Edit: backend/config/contacts.json
   - Add phone numbers for WhatsApp
